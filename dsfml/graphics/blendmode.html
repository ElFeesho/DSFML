dsfml.graphics.blendmode



enum BlendMode: int;
Available blending modes for drawing.

See Also:


http:
//www.sfml-dev.org/documentation/2.0/group__graphics.php#ga80c52fe2f7050d7f7573b7ed3c995388

Authors:
Laurent Gomila, Jeremy DeHaan

Alpha
Pixel = Source * Source.a + Dest * (1 - Source.a)

Add
Pixel = Source + Dest.

Multiply
Pixel = Source * Dest.

None
Pixel = Source.
